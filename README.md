Reflection
How can I ensure that my code, program, or software is functional and secure?

I ensure functionality and security by writing comprehensive unit tests and validating inputs according to requirements. In the Contact Service project, I used JUnit testing to confirm that all fields followed length and null restrictions. Automated testing helps detect errors early and ensures consistent behavior after changes.

How do I interpret user needs and incorporate them into a program?

I analyze requirements carefully and translate them into specific validation rules and test cases. By writing tests directly from requirements, I ensure the program behaves exactly as users expect.

How do I approach designing software?

I design software by separating responsibilities into focused classes and following object-oriented principles. I prioritize readability, modularity, and testability to ensure the system is maintainable and scalable.
